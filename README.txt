== Description ==

A plugin to talk online with the other users of a blog.

== Installation ==

1. Copy the 'chat' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag and drop the 'Chat' widget to any sidebar on the 'Appearance > Widgets' menu in WordPress

== Usage ==
1. Login to WordPress
2. Go to the page where de chat widget is
3. Click on the name of the user you want to talk to open the chat dialog
4. Start chating
